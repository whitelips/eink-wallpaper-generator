Version 1.6.0 2025-06-21

⚖️ 비례 크기 조정 시스템

  • 모든 구성 요소에 대한 비례 크기 조정 시스템 구현
  • 최소/최대 크기 간 선형 보간 적용
  • 기기 높이 기반 크기 조정 (1440px ~ 3200px)
  • 1.0x ~ 2.0x 비례 크기 조정 범위
  • 제목 최대 180px, 아티스트 최대 130px로 제한
  • 모든 해상도에서 부드러운 전환 효과

📝 텍스트 가독성 개선

  • 모든 텍스트 글꼴 크기 20% 증가
  • 가사: 높이 > 2000px일 때 180% 크기
  • 시간 레이블: 높이 > 1800px일 때 150% 크기
  • 진행률 표시줄: 높이 > 1800px일 때 180% 높이

📚 문서화

  • 변경 로그 하위 페이지 추가
  • 한국어/영어 이중 언어 지원
  • 홈페이지 하단에 변경 로그 링크 추가
  • GitHub 링크 제거로 간소화
  • 변경 로그 날짜 수정 (6월 2025)
